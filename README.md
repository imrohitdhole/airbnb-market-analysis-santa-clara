# Airbnb Market Analysis — Santa Clara County

Market analysis of Airbnb listings in Santa Clara County to identify pricing drivers, high-demand areas, and seasonal patterns.

## What’s inside
- `Airbnb.ipynb` — data cleaning, EDA, geospatial analysis, and modeling
- `report/Group6_DV_ProjectReport.pdf` — full project report with findings and visuals

## Highlights
- Web scraping / data collection for 15K+ listings
- Geospatial analysis to identify high-demand neighborhoods and pricing zones
- Predictive pricing model incorporating seasonality and property features
- Dashboard-style visuals for market insights

## Tech
Python, pandas (plus geospatial + visualization libraries)

## How to run
1. Open `Airbnb.ipynb` in Jupyter/Colab
2. Install required packages (if prompted)
3. Run cells top-to-bottom
