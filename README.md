# Airbnb Market Analysis — Santa Clara County

Market analysis of Airbnb listings in Santa Clara County to identify pricing drivers, high-demand areas, and seasonal patterns.

## Demo
🎥 YouTube walkthrough: https://youtu.be/4LTOOuzI07E

[![Project Demo](https://img.youtube.com/vi/4LTOOuzI07E/0.jpg)](https://youtu.be/4LTOOuzI07E)

## What’s inside
- `notebooks/Airbnb.ipynb` — data cleaning, EDA, geospatial analysis, and modeling
- `report/Group6_DV_ProjectReport.pdf` — full project report with findings and visuals

## Highlights
- Geospatial analysis to identify high-demand neighborhoods and pricing zones
- Pricing/demand insights using seasonality and listing features
- Dashboard-style visualizations for market insights

## Tech
Python, pandas, visualization + geospatial libraries

## How to run
1. Open `notebooks/Airbnb.ipynb` in Jupyter or Google Colab
2. Install required packages if prompted
3. Run the notebook top-to-bottom
